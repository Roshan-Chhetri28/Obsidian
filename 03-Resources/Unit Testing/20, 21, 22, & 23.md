## Testing For Errors:
- we would want to add test for cases where function should throw error 
- we can do something like the following:
```js
it('should throw an error if no value is passed to the function', ()=>{
	const resultFn = () => {
		add()
	}
	
	//we expect it to throw error
	expect(resultFn).toThrow()
}) 
 
 
npm run test -> test passed
```
- If we expect the code to not throw and we want to cover the case then we can use ==not== keyword in following way
 ```js
it('should not throw an error if no value is passed to the function', ()=>{
	const resultFn = () => {
		add()
	}
	
	//we expect it to throw error
	expect(resultFn).not.toThrow()
}) 

npm run test -> the test doesn't pass
 ```
